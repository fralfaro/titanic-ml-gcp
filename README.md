# Titanic ML Pipeline en GCP

Este repositorio contiene una **implementación de referencia de un pipeline de Machine Learning end-to-end en Google Cloud Platform (GCP)**.

El dataset Titanic se utiliza de forma intencional como un **ejemplo simple**, con el objetivo de poner el foco en **buenas prácticas de ingeniería, reproducibilidad y MLOps**, más que en la complejidad del modelo.

## Objetivo del proyecto

- Diseñar un pipeline de Machine Learning reproducible.
- Separar claramente las etapas de datos, entrenamiento y despliegue.
- Utilizar herramientas modernas del ecosistema Python y GCP.
- Documentar decisiones técnicas de forma clara.
